# 🚀 About

**Snake Cyber Game** is a browser-based remake of the classic Snake game with a futuristic cyberpunk style.

Instead of the traditional appearance, the game features a neon-inspired interface, glowing visual effects, dynamic speed progression, and responsive controls for both desktop and mobile devices.

The project focuses on delivering a modern gameplay experience while preserving the simplicity and excitement of the original Snake game.

---

# ✨ Features

* 🐍 Classic Snake gameplay
* 🌌 Futuristic cyberpunk design
* 💡 Neon visual effects
* 🏆 Score & Best Score system
* ⚡ Progressive speed increase
* 📱 Mobile touch controls
* 👆 Swipe gesture support
* ⌨️ Keyboard controls (Arrow Keys & WASD)
* ⏸️ Pause functionality
* 💾 Automatic high-score saving
* 🎨 Smooth animations
* 🌐 Play directly in the browser

---

# 🎮 Gameplay

Control the snake, collect food, increase your score, and avoid collisions with walls or your own body.

As your score increases, the game becomes faster, making every new point more challenging than the last.
